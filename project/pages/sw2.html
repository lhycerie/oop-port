Conditional Statements

➤ March 2026

Explanation & Learnings

In this seatwork, we were given a problem that required us to make a Java program that utilized conditional statements, such as if-else and switch-case. The task was a lot harder as coding in paper is really different from coding in an IDE, since I can't see the errors right away. The program is about making an ATM machine that can perform basic banking operations like checking balance, depositing money, and withdrawing money.

To be honest, I found this quite challenging, as my skills in programming is more focused on front-end development. With me still studying and having limited experience in Java, the syntax was a lot more new to me, I had to really think through the logic of the program and how to structure the conditional statements to achieve the desired output.
