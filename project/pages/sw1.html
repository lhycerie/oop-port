Seatwork 1

Increment & Decrement

➤ March 2026

Explanation & Learnings

In this seatwork, the topic was about increment and decrement operators in Java. The concept is really just the same as C so, it was easier to grasp. However, the problems given can get really complex when there are a lot of incrementing involved. I had to analyze a Java program to determine the output, which involved tracing the changes in variable values as the program executed. This exercise helped me understand how pre-increment and post-increment operators work like "++a" and "a++", as well as how they affect the flow of the program. It was interesting to see how the order of operations can lead to different results, and it reinforced the importance of understanding operator precedence in programming.
